# Bank Customer Transaction Analysis 🏦

## Overview
SQL-based analysis of bank customer transactions to identify trends, 
detect fraud, and segment customers by value.

## Dataset
- 40 transactions
- 10 customers
- 3 months data (Jan-Mar 2024)
- Fields: Transaction ID, Customer, City, Branch, Date, Type, Amount

## Analyses Performed
1. View all transactions
2. Top 5 customers by total amount
3. Monthly transaction trends
4. Customer wise monthly transactions
5. Fraud detection — suspicious high value transactions
6. Customer segmentation — High/Medium/Low value
7. Branch wise performance
8. Credit vs Debit analysis
9. City wise analysis
10. Highest single transaction per customer
11. Inactive customers detection
12. Average transaction amount per branch

## Tools Used
- MySQL
- DB Fiddle (online SQL editor)
- GitHub

## Key Findings
- Ravi Verma is highest value customer
- Fraud detected in 6 transactions above 50,000
- Credit transactions higher than Debit in total volume
